# Blackjack!

A simple command line blackjack game written in NodeJS.

### Features
- [x] Play multiple rounds
- [x] Deal cards from a deck
- [x] 'Hit' until you have a winning hand (or bust)
- [x] Compete against the dealer
- [ ] Bet on hands
- [ ] Play with multiple players
- [ ] Reshuffle deck
- [ ] Lots of other stuff
